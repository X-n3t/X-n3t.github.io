    Escalate Privileges - www-data => root

    • Se observa una plantilla de un script en python, y luego en el directorio "bin" el binario con privilegios de suid:

      www-data@bank:/var/htb$ cat emergency 
#!/usr/bin/python
import os, sys

def close():
    print "Bye"
    sys.exit()

def getroot():
    try:
        print "Popping up root shell..";
        os.system("/var/htb/bin/emergency")
        close()
    except:
        sys.exit()

q1 = raw_input("[!] Do you want to get a root shell? (THIS SCRIPT IS FOR EMERGENCY ONLY) [y/n]: ");

if q1 == "y" or q1 == "yes":
    getroot()
else:
    close()
